For production specific setup, see developer docs > production.

Quizfreely's website/webapp ("quizfreely-web") is written in JavaScript with Node.js, Fastify (web framework/server), and Eta (template engine). Quizfreely's api or backend ("quizfreely-api") fetches data from our PostgreSQL database. Quizfreely-API has a GraphQL API and a REST API, it's written in Javascript with Node.js, Fastify, and Mercurius (GraphQL adapter for Fastify).

Quizfreely is free and open source. It's source code is on Codeberg and GitHub. These are NOT mirrors; pull requests and issues are welcome on both platforms.
