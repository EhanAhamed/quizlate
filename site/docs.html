Saving study sets

As of right now, Quizfreely does not have an account system.

Accounts with auto save and other fancy features is a planned feature and will be added soon. Please see the about page for more details!


Right now, Quizfreely can save your data as local files! This is the only saving option until accounts and auto save is added.

When you save your study sets as local files, they are only accessible on the device they are downloaded on.

To be able to save, access, and share your study sets across multiple devices, you should save them to a cloud storage account, like Google Drive.


Quizfreely's local files are just like any other file you download on the internet. You can save them to any cloud storage you want. If you need help using a specific cloud storage, see that specific site's help page. Google Drive's help page is linked here.
